# ChemSphere Nexus (Pinpools Clone)

**The Operating System for Agile Chemical Procurement.**

ChemSphere Nexus is a B2B platform for streamlined chemical sourcing, real-time intelligence, and verified marketplace transactions.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Database:** SQLite (Dev) / PostgreSQL (Prod) via Prisma ORM
- **Styling:** Tailwind CSS + Shadcn UI
- **Auth:** NextAuth v5 (Auth.js)

## Getting Started

1. `npm install`
2. `npx prisma db push`
3. `npm run dev`
